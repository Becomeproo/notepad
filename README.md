## ℹ️앱 설명

단어장 앱으로 단어를 추가, 삭제 및 수정을 할 수 있는 앱이다.

<br>
<br>

## ✅구현 기능

* 단어장 UI 구현
* 단어 추가
* 단어 수정
* 단어 삭제

<br>
<br>

## ✅사용되는 기능

* UI
  * RecyclerView, Adapter
  * TextInputLayout, TextInputEditText
  * ChipGroup, Chip

* Kotlin
  * data class

* Android
  * Room
  * registerForActivityResult
  * Parcelize
